# Aggregated Analysis

Generated by `scripts/build_aggregated_ridership_parquet.py`.

These files are concatenated Parquet copies of the older route-level APC Excel
reports in `data/raw/TCAT_Ridership_Aggregated/`. They are kept for legacy
comparison with the old hour-aggregated analysis, not as the preferred DiDisc
modeling layer.

## Files

- `apc_by_hour.parquet`: concatenated APC-by-hour reports.
- `apc_by_trip.parquet`: concatenated APC-by-trip reports.
- `apc_by_stop.parquet`: concatenated APC-by-stop reports.
- `apc_by_month.parquet`: concatenated APC-by-month reports.
- `apc_by_dotw.parquet`: concatenated APC-by-day-of-week reports.
- `aggregated_parquet_manifest.csv`: row counts and file sizes.
