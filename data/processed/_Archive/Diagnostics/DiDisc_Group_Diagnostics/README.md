# Difference-in-Discontinuities Group Diagnostics

Generated by `scripts/_archive_diagnostics/didisc_group_diagnostics.py`.

These files support treatment/control definitions for Cornell's free
weekday-after-6 pm and weekend policy.

Recommended interpretation:

- Treatment is Cornell exposure at the route or route-stop level.
- `Second Left Arrow 16` is an override/compliance signal, not a treated group.
- `Cornell Card` is an exposure proxy or placebo group, not the main control,
  because these riders already have unlimited access.

Important outputs:

- `route_near_6pm_summary.csv`: pre/post 6 pm route evidence.
- `route_cornell_exposure_summary.csv`: route fare composition by period.
- `stop_cornell_exposure_summary.csv`: stop-level pre-6 Cornell Card exposure.
- `candidate_route_set_diagnostics.csv`: checks for candidate treated/control
  route sets.
- `fare_group_by_policy_period.csv`, `fare_family_by_policy_period.csv`, and
  `fare_category_by_policy_period.csv`: fare mix across paid/free-policy periods.
