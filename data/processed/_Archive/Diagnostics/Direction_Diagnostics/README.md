# Direction Diagnostics

Generated by `scripts/_archive_diagnostics/investigate_direction_column.py`.

This folder documents the interpretation of vehicle-history direction fields.

Conclusion:

- `direction_vhist` is a compass heading/bearing, not a stop ID.
- `Inbound_Outbound_vhist` is a route-specific categorical direction code, not a
  stop ID.
- Use `Stop_Id_vhist`, `Previous_Stop_Id_vhist`, and `Stop_Name_vhist` for stop
  information.

Important outputs:

- `direction_column_summary.csv`: summary tests for direction versus stop IDs.
- `direction_value_counts.csv`: observed bearing values.
- `inbound_outbound_by_route.csv`: route-level direction-code distribution.
- `direction_sequence_sample.csv`: ordered examples showing direction changes
  like headings rather than stop IDs.
