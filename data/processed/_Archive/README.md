# Processed Data Archive

This folder keeps reproducibility and audit files out of the active processed
data root.

## Diagnostics

`Diagnostics/` contains repeatable diagnostic outputs for missing/invalid
records, treatment/control group selection, and direction-field interpretation.
They support documentation and writeup decisions, but they are not modeling
inputs.

## Exploration

`Exploration/` contains older one-off notebook outputs. They are preserved for
history only.
